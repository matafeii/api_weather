# Weather App

Погодное приложение с анимациями и интерактивной картой.

## Структура проекта

```
api_weather/
├── index.html          # Главная HTML страница
├── styles.css         # Стили приложения
├── package.json       # Конфигурация npm (для сборки с Vite)
├── js/
│   ├── main.js        # Точка входа
│   ├── weather.js     # Логика погоды и API
│   ├── map.js         # Интерактивная карта (Leaflet)
│   ├── scene.js       # 3D сцена (Three.js)
│   └── animation.js   # GSAP анимации
└── assets/
    ├── textures/      # Текстуры для 3D
    └── models/        # 3D модели
```

## Функции

- Отображение текущей погоды для польских городов
- Выбор города на интерактивной карте (Leaflet)
- GPS определение местоположения
- Прогноз погоды на 5 часов
- Анимированные эффекты погоды (солнце, дождь, снег, облака, молнии, туман)
- 3D частицы с использованием Three.js и GLSL шейдеров
- Плавные анимации с GSAP

## Запуск

### Вариант 1: Без сборки (открыть index.html)

Просто откройте `index.html` в браузере.

### Вариант 2: С использованием Vite (рекомендуется)

```
bash
# Установка зависимостей
npm install

# Запуск dev сервера
npm run dev

# Сборка для продакшена
npm run build
```

## API

Приложение использует бесплатный API [Open-Meteo](https://open-meteo.com/) для получения данных о погоде.
